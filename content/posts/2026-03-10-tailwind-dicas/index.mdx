O Tailwind permite estilizar componentes sem sair do arquivo HTML/JSX. A produtividade é incomparável quando combinado com um design system bem estruturado — classes utilitárias eliminam a alternância entre arquivos CSS e componentes, mantendo o contexto visual sempre próximo do código.

  variant?: 'default' | 'outline' | 'ghost'

  variant = 'default',

## Steps

<Steps>
  <Step title='Instale as dependências'>
    Adicione o Tailwind e o plugin PostCSS ao projeto:

    ```bash
    bun add tailwindcss @tailwindcss/postcss
    ```

  </Step>

  <Step title='Configure o PostCSS'>
    Crie ou edite o arquivo `postcss.config.mjs` na raiz do projeto:

    ```js title="postcss.config.mjs"
    export default {
      plugins: {
        '@tailwindcss/postcss': {},
      },
    }
    ```

  </Step>

  <Step title='Importe no CSS global'>
    Adicione a diretiva de importação no topo do seu arquivo CSS principal:

    ```css title="globals.css"
    @import 'tailwindcss';
    ```

  </Step>

  <Step title='Use no projeto'>
    Agora você já pode usar as classes utilitárias nos seus componentes:

    ```tsx
    export function Card() {
      return (
        <div className='rounded-lg border bg-card p-6 shadow-sm'>
          <h2 className='text-lg font-semibold'>Pronto!</h2>
        </div>
      )
    }
    ```

  </Step>
</Steps>
